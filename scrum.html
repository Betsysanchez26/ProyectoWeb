5 Utilizar un lenguaje de programación del lado del servidor para la construcción de aplicaciones Web. En este objetivo se utilizára JavaScrip para la comunicación del cliente con el servidor, el cual se basa principalmente en la creación de la base de datos, así como las posteriores peticiones que se requieran para el correcto funcionamiento de la página, enfocado a las peticiones del cliente. 80%
